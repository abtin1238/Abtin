# تاریخچه تغییرات (Changelog)

## نسخه 0.1.0+1 - فاز ۳ (جدید) ✨

### ✅ اضافه شده
- **سرویس مسیریابی OSRM**
  - فایل جدید: `lib/features/routing/data/routing_service.dart`
  - محاسبه مسیر بین دو نقطه با API رایگان OSRM
  - ترجمه خودکار دستورات به فارسی
  - محاسبه فاصله و زمان تقریبی

- **مدیریت مسیریابی فعال**
  - فایل جدید: `lib/features/routing/presentation/routing_providers.dart`
  - Provider برای مسیریابی با Riverpod
  - پیگیری پیشرفت در مسیر
  - مدیریت دستورات پیچ‌به‌پیچ

- **نمایش مسیر روی نقشه**
  - رسم Polyline مسیر با رنگ سبز
  - حذف خودکار مسیر قبلی
  - انیمیشن روان برای ترسیم

- **رابط کاربری مسیریابی**
  - کارت مسیریابی فعال با دستورات Turn-by-Turn
  - نمایش فاصله و زمان باقیمانده
  - آیکون‌های مناسب برای هر نوع دستور
  - دکمه "پایان مسیریابی"
  - اعلان رسیدن به مقصد

- **محاسبات مسیریابی**
  - تشخیص موقعیت فعلی در مسیر
  - به‌روزرسانی دستور فعلی بر اساس فاصله
  - محاسبه فاصله با فرمول Haversine
  - تشخیص خودکار رسیدن به مقصد (آستانه ۲۰ متر)

### 🔄 تغییر یافته
- `home_screen.dart`: اضافه شدن منطق مسیریابی کامل
- `pubspec.yaml`: اضافه شدن پکیج `http: ^1.2.0`
- `README.md`: به‌روزرسانی مستندات
- دکمه "شروع مسیریابی" فعال شد (دیگر placeholder نیست)

### 🐛 رفع شده
- دکمه "شروع مسیریابی (بزودی)" اکنون کاملاً کار می‌کند
- مسیر واقعی روی نقشه نمایش داده می‌شود

## نسخه 0.1.0 - فاز ۲

### ✅ اضافه شده
- **GPS واقعی**
  - `LocationPermissionFlow` برای دریافت مجوز
  - دیالوگ خودکار برای روشن کردن GPS
  - پیگیری موقعیت real-time

- **هموارسازی GPS**
  - `LocationService` با فیلتر Exponential Moving Average
  - هموارسازی موقعیت و Heading
  - کاهش پرش و لرزش

- **مارکر خودرو**
  - حالت پیش‌فرض: پیکان
  - حالت خودرو: BMW i8 (آیکون دوبعدی)
  - چرخش بر اساس Heading

- **مسیریابی با لمس نقشه**
  - انتخاب مقصد با تپ روی نقشه
  - نمایش Pin مقصد
  - کارت اطلاعات مقصد

- **Deep Linking**
  - `DeepLinkService` با پکیج app_links
  - پشتیبانی از `abtin://navigate?lat=..&lng=..&label=..`
  - عملکرد در Cold Start و Hot State

- **دیتابیس محلی**
  - جدول SavedPlaces با Drift (SQLite)
  - صفحه علاقه‌مندی‌ها
  - CRUD کامل

### 🔄 تغییر یافته
- به‌روزرسانی AndroidManifest برای مجوزها و deep link
